# Hub SGI Lite

**Sistema de Gestão Integrado — Versão independente de API**

Uma plataforma completa para implantação e manutenção de sistemas de gestão integrados conforme **ISO 14001** e **ISO 45001**, com funcionalidade 100% local — sem necessidade de chave de API, sem custo por uso, funciona offline.

## Características

- ✅ **100% local** — nenhum dado sai do navegador
- ✅ **Funciona offline** após o primeiro carregamento
- ✅ **Sem custo de API** — análise crítica baseada nos dados preenchidos
- ✅ **Dark mode** — alterna entre claro e escuro, persiste entre sessões
- ✅ **Parametrização de normas** — configure quais normas sua organização possui
- ✅ **Exportação** — gere evidências para auditoria em PDF
- ✅ **Formulário de campo** — levantamento offline com exportação JSON
- ✅ **ISO 9001** — previsto para set/2026

## Como usar

1. Acesse `index.html` no navegador
2. Clique em **Contexto da Organização** para começar
3. Preencha os módulos em sequência
4. Use **Análise Crítica** para gerar o relatório da Alta Direção
5. Exporte as evidências para auditoria

## Tecnologia

HTML + CSS + JavaScript puro. Sem dependências externas, sem backend, sem banco de dados.
Os dados são salvos automaticamente no `localStorage` do navegador.
